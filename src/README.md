# Dynamic Load Balancing in Cloud Using Swarm Intelligence

This project simulates dynamic load balancing using PSO, Round Robin, and Random strategies.

## How to Run
1. Create a virtual environment:
   python3 -m venv venv

2. Activate it:
   source venv/bin/activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run the project:
   python3 src/main.py
